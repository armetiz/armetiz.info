Pour résumé, Oracle c’est :

  • Le géant de la base de donnée.
  • Un contributeur pour le libre, et le 4° plus gros contributeur du noyau Linux

Et pour SUN on a :

  • Un système d’exploitation avec Solaris.
  • Une machine virtuelle avec JVM.
  • Un langage de programmation Java qui marque les bases de nombreux autres langages.
  • L’inventeur de NFS qui est un système de fichiers en réseau très répandu sous LINUX.
  • Mais aussi, le propriétaire de MySQL.
  • Le propriétaire de VirtualBox.
  • Une plainte historique contre Microsoft pour dénoncer le manque d’interopérabilité de l’Active Direcotry.
  • Le papa de ZFS que l’on aurai bien vu continuer d’être développé.

Et dans le document publié sur le site d’Oracle, il n’est mention que de Java et Solaris, mais pas de MySQL ou de VirtualBox.
Que va-t’il advenir du soutient de SUN envers le monde libre ?

Le montant de la transaction est au final de 7.4 Milliards de Dollars, soit 9.5 Dollars par action. On remarquera que IBM avait fait une proposition à 9.4 Dollars par action qui avait été rejetée deux semaines auparavant.
