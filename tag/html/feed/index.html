L’exemple d’utilisation qui me vient à l’esprit, c’est l’affichage d’un fichier PDF.

Personnellement, cela me sert pour afficher mon CV sur thomas.tourlourat.com

Pour que le PDF utilise toute la zone du navigateur, il faut utiliser la petite astuce CSS suivante :

1
2
3
4

6
7
html,body
{
   width:100%;
   height:100%;
   margin:0px;
   padding:0px;
}

Afficher ensuite votre iFrame

1
2
3
4
5
6

<iframe
        src="http://tourlourat.com/documents/cv_tourlourat_thomas.pdf"
        id="cv"
        width="100%"
        height="100%"
        scrolling="auto"
        frameborder="0" >
</iframe>

Cela est pratique pour faire défiler des images, ou des blocs contenus dans des div.

Scrollable

Scrollable est un tools de la suite Flowplayer. Il est rapide à mettre en place et utilise JQuery. La documentation est suffisamment fournit pour utiliser correctement le script.
Les options de configuration sont assez nombreuses.

Horinaja

Horinaja utilise JQuery ou scriptaculous, ce sera à vous de choisir la version que vous souhaitez.
La documentation semble plus petite, mais est-ce un tord ? A vous de jugez car je n’ai pas encore utilisé cette solution.
