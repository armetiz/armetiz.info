C’est le but du concours 25lines.com, obtenir des animations intéressantes en un nombre de ligne de code réduit.

Le concours a commencé il y a environ un mois, et les développeurs ont eu le droit d’utiliser le player version 10, on retrouvera donc pas mal de réalisation en 3D.

Il vous suffit de vous inscrire pour élire votre finaliste préféré.
Personnellement, j’ai voté pour l’animation 55, mais j’ai beaucoup hésité avec la numéro 43.

Quand on voit ce que certains peuvent faire en 25 lignes de code, on se dit qu’il y a encore beaucoup à apprendre.
